# claude-review

A CLI tool and GitHub Action for automated PR review using Claude Code.

## Install

```bash
npm install -g claude-review
```

Or build from source:

```bash
git clone https://github.com/phoenix7956/claude-review.git
cd claude-review
npm install
npm run build
npm link
```

## Usage

### CLI

```bash
export GITHUB_TOKEN=your_github_token
claude-review https://github.com/owner/repo/pull/123
```

### GitHub Action

See [.github/workflows/review.yml](.github/workflows/review.yml) for the workflow configuration.

## How It Works

1. Fetches the PR diff via GitHub API
2. Analyzes the diff with Claude Code (or pattern-based fallback)
3. Posts a structured Markdown review comment

## Output Format

```markdown
## 📋 PR Review: <PR Title>

### Summary
<2-3 sentence summary of changes>

### 🔍 Risks
- <list of identified risks>

### 💡 Suggestions
- <list of improvement suggestions>

### ✅ Confidence Score: **Low / Medium / High**

---
*Analyzed <N> file(s) | <+X> additions / <−Y> deletions*
*Review generated by claude-review CLI*
```
